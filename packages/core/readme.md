# @bilibili-dl/core

Core package for Bilibili-DL Project

## Installation

Install `@bilibili-dl/core`

```bash
  npm install @bilibili-dl/core
```

or

```bash
  pnpm add @bilibili-dl/core
```

or

```bash
  yarn add @bilibili-dl/core
```
